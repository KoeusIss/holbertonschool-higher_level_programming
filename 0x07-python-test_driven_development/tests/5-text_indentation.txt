The ``text_indentation`` module
===============================================================================

Using ``text_indentation``
-------------------------------------------------------------------------------

Import text_indentation from 5-text_indentation

    >>> text_indentation = __import__('5-text_indentation').text_indentation

It should raise an exception if 'text' is not a string

    >>> text_indentation(98)
    Traceback (most recent call last):
        ...
    TypeError: text must be a string

It should print nothing if the 'text' is empty

    >>> text_indentation("")

It should tokenize a given string based on '.', ':' and '?' seperator

    >>> text_indentation("hello world. from the other side of the moon\
    ... we sent you our greeting? do you enjoyit or: sad, happy, anything.")
    hello world.
    from the other side of the moon we sent you our greeting?
    do you enjoyit or:
    sad, happy, anythong.
