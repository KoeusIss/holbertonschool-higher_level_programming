The ``MyList`` module
===============================================================================

Using ``print_sorted``
-------------------------------------------------------------------------------


Import `print_sorted` from `1-my_list`

    >>> MyList = __import__('1-my_list').MyList

It should create successfully an instance of MyList

    >>> lst = MyList()

It should append a value to the lst

    >>> lst = MyList()
    >>> lst.append(1)

it should print the right form

    >>> lst = MyList()
    >>> lst.append(4)
    >>> lst.append(3)
    >>> lst.append(1)
    >>> lst.append(5)
    >>> lst.append(2)
    >>> lst.print_sorted()
    [1, 2, 3, 4, 5]

it should conserve the original list

    >>> lst = MyList()
    >>> lst.append(4)
    >>> lst.append(3)
    >>> lst.append(1)
    >>> lst.append(5)
    >>> lst.append(2)
    >>> lst.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(lst)
    [4, 3, 1, 5, 2]
